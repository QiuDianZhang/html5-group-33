# html5-group-33
html5个人书单微信小程序作业
##  1: 实现的功能

|               |                                          |
| ------------- | ---------------------------------------- |
| 1. 书籍录入   | 手动输入书籍信息（弹窗/页面）            |
| 2. 书库展示   | 列表/网格布局展示所有书籍                |
|               | 按状态筛选（未读/想读/不想读/在读/已读） |
|               | 下拉刷新                                 |
| 3. 书籍管理   | 查看书籍详情                             |
|               | 编辑书籍信息                             |
|               | 删除书籍                                 |
| 4. 数据统计   | 首页显示书籍总数、各状态数量             |
| 5. 数据持久化 | 数据保存到本地                           |
| 6. 设置       | 清空所有数据                             |

## 2: 关键技术

WXML、WXSS、JavaScript、小程序生命周期函数、小程序API  



## 3: 使用工具

微信开发者工具
